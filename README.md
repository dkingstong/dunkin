# Dunkin Donuts dashboard

# frontend
To start the client <br>
1.- Run npm ci from the root directory <br>
2.- Run npm start

# backend
To start the server <br>
1.-run npm ci from the root directory <br>
2.- Run API_KEY={your method API key} npm start <br>
